Themes for Throttlr

Community-made color schemes you can drop into Throttlr. Each one is a small JSON file that re-skins the app and overlay. Pick one, download, drop it in your themes folder, click apply.

Free + open source One-click download Submit your own via PR
↓

How to install

  1. Click Download on a theme card
  2. Extract the ZIP — you get theme.json + theme.css
  3. In Throttlr: Settings → Appearance → Open themes folder
  4. Drop both files in
  5. Click Rescan → click your new theme tile

// contribute

Make a theme,
submit it.

Themes are just JSON files in the themes/ folder of the Throttlr GitHub repo. Fork the repo, add your file, open a pull request. Once it's merged it shows up here automatically — no website redeploy needed.

Your theme can override as much or as little as you want. Set just an accent color and let the rest inherit from a base theme, or override every color for full control.

Read the format spec Browse on GitHub 
{
  "name": "My Theme",
  "author": "@yourname",
  "version": "1.0.0",
  "description": "What it looks like",
  // inherit from one of the built-ins
  "base": "midnight",
  // the swatch shown on the theme card
  "preview": "#ff66cc",
  // override only what you want
  "colors": {
    "bg":     "#1a0a2a",
    "accent": "#ff66cc",
    "text":   "#e0d8f0"
  }
}
