    A profile is your full Throttlr setup — target apps, all 6 function settings, custom presets, filter script, and theme — saved as a single .throttlr file. Useful for backing up your configuration, syncing between PCs, or sharing with friends.

    • Target applications (by name + path)
    • All 6 function settings (Lag / Drop / Throttle / Freeze / Block / Fun)
    • Direction settings (in / out for each function)
    • Custom presets you've saved
    • Filter Script
    • Theme + accent color

    Hotkeys, sound preferences, and overlay layout are not included — those are personal preferences, not part of a profile.

    You can also drag a .throttlr file directly onto the Throttlr window to import it instantly — no need to open this menu.

    Rules fire automatically when their condition becomes true. Edge-triggered — they fire once per condition transition, not continuously.
